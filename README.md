# Startup Guide

1. Initialize user secrets:

   ```bash
   dotnet user-secrets init
   ```

2. Set the Google Maps API key:

   ```bash
   dotnet user-secrets set "GoogleMapsApiKey" "YOUR_API_KEY"
   ```
